Testando uma API Rest

Ultimamente no serviço, venho percebido muita gente com dúvidas ou fazendo confusão sobre os métodos REST. Dúvidas como quando usar os métodos POST, GET e PUT.
Mas antes de usar os métodos é importante entender o que é a tecnologia REST.


REST - Representational State Transfer

Representational State Transfer ou Transferência de Estado Representacional, é um conceito ou modelo arquitetural com conjunto de métodos e regras. Existe muita confusão sobre o uso do REST em relação ao uso do RPC ou Remote Procedure Call.
Apesar dos 2 utilizarem o protocolo HTTP, no RPC, dá-se ênfase à diversidade de operações(getUser, addUser), já no REST, a ênfase está na diversidade de recursos(User, Address) .
