Java + Visual Studio Code.

Não é de hoje que a Microsoft vem inovando em novas tecnologias e soluções.
Computação nas nuvens, IDE's, dispositivos, entre outras coisas.

A 1 ano venho usado a IDE da Microsoft Visual Studio Code ou simplesmente VS Code para programar em Angular.

Por ser uma IDE praticamente recente, as atualizações são frequentes.


Baixando VS Code e extensão Java.

Segue o link para baixar o Visual Studio Code para sua plataforma(Linux, Windows ou Mac).

Instalando as extensões Java:

- No seu VS Code pressione CTRL + Shift + X para abrir a visualização de extensões.
- Digite vscode-java-pack e baixe a extensão Java Extension Pack.


Pode ser que leve alguns minutos para instalar, e o status fica como "Installing" sem barra de progresso o que atrapalha na previsão de instalação. Quando finalizar a instalação, reinicie a IDE.
