Flask-JWT.

O Flask-JWT é a abstração do JWT(Json Web Token) encapsulado em um pequeno módulo para ser utilizado no Micro-servidor Flask.

O JWT é uma forma de garantir integridade dos dados a partir de uma autenticação baseada em token.

O token JWT é composto por 3 componentes separados por “.”: Header, Payload e Signature.

O Header é onde definimos qual tipo de token e que tipo de criptografia é utilizado na composição.

O Payload são as informações que serão utilizados para algum tratamento após a descriptografia.

O Signature é a chave secreta que será composta no token.

Este Post não entrará em detalhes como funciona o JWT, a abordagem será para mostrar uma implementação JWT e Flask.

Implementação Flask+JWT.

Primeiro precisamos fazer a instalação do Flask e do Flask-JWT.

Instalação do Flask:
$ sudo pip install Flask

Instalação do Flask-JWT:
$ sudo pip install Flask-JWT

Rode a aplicação e faça uma chamada via curl para gerar uma chave JWT:
curl -H "Content-Type: application/json"
-d '{"username":"user2", "password": "abcxyz"}'
-X POST http://127.0.0.1:5000/login

Se tudo deu certo é gerado uma chave: {
"access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.
eyJpZGVudGl0eSI6MiwiaWF0IjoxNTY3OTcwNDMxLCJuYmYiOjE1Njc5NzA0MzEsImV4cCI6MTU2Nzk3MDczMX0.
AahLjtEo9QKf31JoceXs7s2IbrqaGk4EeNcNb-Xf3jo"
}

Faça a chamada agora para verificar se a chave é válida:
curl -H 'Accept: application/json'
-H "Authorization: JWT eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.
eyJpZGVudGl0eSI6MiwiaWF0IjoxNTY3OTcwNDMxLCJuYmYiOjE1Njc5NzA0MzEsImV4cCI6MTU2Nzk3MDczMX0.
AahLjtEo9QKf31JoceXs7s2IbrqaGk4EeNcNb-Xf3jo"
http://127.0.0.1:5000/protected

Se tudo deu certo, é retornado o id do objeto: User(id='2')

Conclusão.

Uma forma simples de utilizar uma autenticação JWT baseada em token em Python utilizando Flask e uma abstração JWT, nesse Post apresentamos uma forma simplista da solução, existe várias configurações para customizar sua aplicação.
