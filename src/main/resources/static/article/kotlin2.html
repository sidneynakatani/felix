Sintaxe Kotlin

Kotlin tem uma sintaxe bem fácil de se escrever. Possibilita a escrita de códigos mais expressivas, tornando o projeto mais limpo.
Por ser uma linguagem concisa, podemos fazer muita coisa com menos linhas de código.
Por ser interoperável com Java, migrações de um sistema Java para Kotlin fica mais fácil de ser feito. Além disso, podemos escrever códigos Javascript com Kotlin.


Package e Import


Assim como Java, a declaração do pacote deve ser feito na primeira linha do código, seguido dos imports.

package meu.pacote.teste;

import org.springframework.boot.SpringApplication;

// ...

Por padrão, alguns pacotes básicos são importados automaticamente.

Pacotes básicos

  • kotlin.io.*

  • kotlin.text.*


Definindo a função


Existem algumas formas de declarar a função:

Sem parâmetro e retorno
fun sayHello(){
      Print("Hello")
}
Com 1 parâmetro Int e retorno Int
fun showParam(a: Int): Int {
      return a
}
A mesma expressão acima em 1 linha
fun showParam(a: Int) = a
Com 1 parâmetro e sem retorno (Unit, representa o void do Java)
fun printParam(a: Int): Unit {
     println("show $a")
}
A mesma expressão acima omitindo Unit
fun printParam(a: Int) {
     println("show $a")
}


Definindo variáveis


Variáveis somente leitura:
Declarando variáveis somente leitura
val size: Int = 1      // declarando com tipo e valor
val size = 2    // omitindo tipo
val c: Int      // tipagem é obrigatória quando não é informado o valor
size = 3      // omitindo tipo e atribuição

Variáveis escrita e leitura:
Declarando variáveis escrita e leitura
var size = 5      // omitindo tipo


Função com condicionais


Função com a condicional if/else
fun maiorValor(a: Int, b: Int): Int {
     if (a > b) {
          return a
     } else {
          return b
     }
}
A mesma função acima em 1 linha, estilo ternário
fun maiorValor(a: Int, b: Int) = if (a > b) a else b


Checagem de tipos e cast automático


Verificando a tipagem
fun getTamanho(any: Any): Int? {
     if (any !is String) return null

          return any.length
}
Cast automático
fun showCast(value: Int){
      println("to Double: "+ value.toDouble())
}


Função com laços for / while / when


Função com a laço for
fun exemploFor() {
     val veiculos = listOf("carro", "ônibus", "trem")
     for (veiculo in veiculos) {
          println(veiculo)
}
Função com a laço while
fun exemploWhile() {
     var indice = 0
     val veiculos = listOf("carro", "ônibus", "trem")
     while (indice < veiculos.size) {
          println(veiculo)
          indice++
}
Função com condicional when
fun cores(codigo: Int): String =
when (codigo) {
     1 -> "Azul"
     2 -> "Verde"
     3 -> "Preto"
}
