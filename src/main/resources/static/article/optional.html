Java Optional.

O Optional é uma classe do Java 8 que vai ajudar a remover vários problemas de condicionais confusas.

Da família Do pacote “util”(java.util.Optional), a classe Optional pode receber ou não valores nulos.

Podemos evitar casos de NullPointerException quando utilizamos a classe Optional.

Outro benefício de se utilizar a classe Optional é deixar seu código mais claro e limpo, pois a proposta reduz a necessidade de utilização de condicionais como if/else.

Lembrando que o Optional ajuda a evitar problemas com os erros nulos, mas não vai eliminar o problema, mas com boas práticas o problema é reduzido quase que 100%.

Optional na prática.

No primeiro exemplo, vamos demonstrar como passar 2 String’s, a primeira nula e a segunda com valor e verificar se os valores estão presentes.

Saída:

false
true

Neste outro exemplo, podemos verificar se existe um valor e manipular o valor via “arrow function”.

Saída:

Size: 6

Agora vamos testar uma condicional que verifica se o valor está nulo e desvia o fluxo para uma condição especificada.

Saída:

Outra funcionalidade legal, agora utilizando um filtro para verificar o valor da entrada.

Saída:

Conclusão.

O Optional ajuda muito a evitar códigos confusos e deixar seu código mais limpo e coeso, a API Java tem várias classes que podemos utilizar para melhorar a codificação, o Optional é uma delas.
