Análise de dados com Python+Pandas.

O Pandas é uma lib open source de alta performance para fazer análise de dados.

Bastante utilizado por comunidades acadêmicas, o Pandas vem ganhando mercado pelo seu uso simples e rapidez em analisar e organizar dados.

Combinado com ferramentas IPython e outras lib’s, o Pandas se torna uma ferramenta muito poderosa e performática para análise de dados para o Python.

Para quem não quer utilizar ou aprender a linguagem R, a combinação Pandas e Python já satisfaz a maioria das capacidades do R.

Instalando Pandas no Ubuntu.

Para instalar o Pandas no Ubuntu vamos utilizar o comando “pip” para facilitar:

$ sudo pip install pandas

Baixando massa de dados.

No site kaggle podemos baixar vários Datasets para os mais diversos tipos de pesquisa e análise.

Para esse artigo estou utilizando os dados da Copa da Rússia de 2018.

Analisando os dados com Python.

Depois que baixou o arquivo CSV no Kaggle, temos que entrar no terminal Python e importar o arquivo pelo Pandas.

>>> import pandas as pd
>>> dataset = pd.read_csv('/tmp/WorldCupMatches.csv', sep=',');

A primeira linha importa a lib do Panda e segunda criamos um dataset e importamos os dados do nosso CSV utilizando “,” como separador.

Primeiro, vamos fazer um teste e verificar se os dados foram carregados corretamente com o método “head” para exibir as 5 primeiras linhas e o cabeçalho:

>>> dataset.head();

Para saber o número de linhas que o arquivo contém é só utilizar o método “count”:

>>> dataset.count();

Para ter informações do arquivo, utilizamos o método “describe”:

>>> dataset.describe();

Agora vamos ver alguns dados referente aos jogos do Brasil:

>>> dataset.loc[dataset['Team'] == 'Brazil']

Vamos gerar um gráfico relacionando o número de gols vs tentativas feitas pelo Brasil:

>>> import matplotlib.pyplot as plt
>>> dataset.loc[dataset['Team'] == 'Brazil'].plot(x='Goal Scored',y='Attempts',kind='bar',
title='Goals x Date',color='r')
>>> plt.show()

A primeira linha importa a lib matplotlib, a segunda linha monta a query para pegar todos os resultados dos jogos do Brasil e relacionar gols marcados vs tentativas e a última linha gera o gráfico.

Conclusão.

O Pandas é uma ótima lib com várias ferramentas para fazer análise e organizar estruturas de dados. Além disso é bem eficiente e performático, fazendo com que seja uma ótima ferramenta para ser usada em produção.
