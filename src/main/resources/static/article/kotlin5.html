Kotlin Server-Side.

Vou mostrar uma aplicação Server-Side utilizando Maven, Spring Boot e Spring Data utilizando banco em memória. Bem simples, somente para mostrar quanto é simples criar uma aplicação provedora de serviço em alguns minutos.
Baixei o projeto Git com toda estrutura necessária do link



Estrutura do Projeto
Observe a estrutura bem simples com um pacote "io.mikael.app" com 3 classes
  • app.kt: classe principal que inicia o serviço;
  • model.kt: classe com 2 data model's;
  • service.kt: classe responsável pelo serviço.
Na pasta resource mais 2 arquivos:
  • application.yml: arquivo com as configurações do sistema;
  • import.sql: script para criação da base de dados;
Na raiz resource 1 arquivo Pom:
  • pom.xml: arquivo para geração das dependências do projeto.
Dependências

Execute o install do Maven no projeto para baixar as dependências.



Iniciando o Serviço

Clique com botão direito na classe app.kt e selecione "run as -> kotlin application".



Executando o Serviço

No seu browser, digite "http://localhost:8080/hello"



Conclusão.

Em alguns minutos levantamos um Server-Side provendo um serviço Rest, mostrando uma forma simples utilizando Spring boot para isso.
