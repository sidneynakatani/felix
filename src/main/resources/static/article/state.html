Design pattern State.

O State é um padrão de projeto comportamental.
O padrão de projeto State é utilizado quando existe a necessidade de alterar o comportamento do objeto quando seu estado é alterado.

Basicamente é criado um objeto que vai controlar o comportamento dos outros objetos observando a alteração do seu estado.

Um exemplo, seria um projeto de um semáforo, onde mudamos a sinalização quando recebemos alguma alteração de estado das cores.
Utilizando o padrão de projeto State podemos criar um contexto que ficará aguardando a alteração de estado para mudar o comportamento do objeto.


Como funciona o State.

O pattern State exibe um design onde trabalhamos com estado dos objetos.

Utilizando um Contexto, controlamos todo comportamento e estado dos objetos.

Uma vez instanciado o Contexto, ele fica responsável em receber os estados e alterar o comportamento dos objetos.


Exemplo na prática(Java).

Vamos criar um exemplo de como implementar o padrão State utilizando Java.

Criamos uma interface Semaphore que vai ser a base dos estados que vamos implementar.



Criamos a classe implementadora GreenLight, com o estado Green.



Criamos a classe implementadora RedLight, com o estado Red.



Criamos a classe implementadora YellowLight, com o estado Yellow.



Criamos a classe de contexto TrafficLightControl que vai controlar o estado e alterar o comportamento dos objetos.

Conforme usuário insere o código, o estado do objeto é alterado.



Executando a aplicação.

Conclusão.

O padrão State pode ser utilizado quando existe um grande número de comportamento a ser controlado.
