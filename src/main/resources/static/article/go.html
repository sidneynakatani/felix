Go Lang.

Conhece a linguagem de programação Go Lang?

O Go é uma linguagem compilada, desenvolvida em 2009 pela Google com proposta de programação concorrente.

Nesse Post não vamos comparar se o Go é melhor ou pior que outras linguagens de programação, mas falar sobre suas vantagens e desvantagens.

A comparação de linguagens de programação hoje em dia é algo totalmente desnecessária, cada linguagem tem sua vantagem para alguma solução.

O Go tem como destaque a simplicidade, performance e facilidade de programação com concorrência.

Vantagens do Go.

  • Performance: Por ser uma linguagem compilada para máquina, ele naturalmente será mais rápida que uma linguagem interpretada;
  • Programação concorrente: O Go tem como proposta a facilidade de programação concorrente.

Desvantagens do Go.

  • Suporte a biblioteca de terceiros: Estamos em 2019, sendo 10 anos de existência e o número de bibliotecas de terceiros ainda é pequena;
  • Gerenciamento de dependência: O Go não tem um gerenciador de dependência eficiente como NPM para Node.js ou PIP para Python.

Instalação no Ubuntu.

Siga as instruções abaixo para instalação do Go no Ubuntu:

Baixe o instalador Google's golang installer para facilitar a nossa instalação:
$ wget -q https://storage.googleapis.com/golang/getgo/installer_linux

Deixe o instalador executável:
$ chmod +x installer_linux

Execute o comando:
$ ./installer_linux

Se a instalação foi executada com sucesso, a seguinte tela será apresentada:

Execute o comando que está em destaque, lembrando que o comando vai ser diferente para cada computador.

Goroutines e Channel.

Entenda Goroutines como threads, em Go, podemos com Goroutines gerar códigos concorrentes com muita facilidade.

Para quem utiliza threads Java, C, C++, entre outras linguagens, vai ficar surpreso com a facilidade de se criar e executar threads com Go.

Repare que para “ativar” a thread é só colocar a função go na frente da função chamadora.

Já o Channel possibilita identificar que a Goroutine terminou a execução.

Conclusão.

Perceba que não existe uma linguagem de programação melhor do que outra, mas sim, existem linguagens que se aplicam melhores em determinadas soluções e com Go, podemos aplicar bem em soluções onde necessitam de utilização de concorrência.
