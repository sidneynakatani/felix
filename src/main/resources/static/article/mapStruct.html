MapStruct.

A Api MapStruct contêm funcionalidades que facilitam o mapeamento entre 2 Java Beans.

Utilizando a Api(MapStruct) podemos converter por exemplo um Dto em uma Entity com facilidade.

A única tarefa que precisamos fazer é criar uma interface e mapear via anotação todos os campos que desejamos transferir de uma Bean para outra.

A Api pode ajudar muito quando precisamos utilizar o pattern “Adapter” onde geralmente precisamos transferir os dados de entrada do endpoint em um Pojo para ser persistido na base de dados.

Geralmente esse processo de criação é feito totalmente manualmente criando uma classe que fica responsável em fazer a transferência de dados. Utilizando MapStruct, essa tarefa trabalhosa fica totalmente a cargo da API.
