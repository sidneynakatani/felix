Introdução ao React.

Mas afinal, o que é o React?
Diferente do Angular que é um Framework onde temos um conjunto de módulos como Routes, Pipes, entre outros, o React é somente uma Lib Javascript para criar interface de usuário.

Por focar somente na camada de apresentação, o React se torna bem eficiênte pois trabalha unicamente nessa camada. Trabalhar somente com Interface de Usuário torna o desenvolvimento mais simples.

Para definir uma solução completa em React precisamos incluir outras bibliotecas. Mas cada biblioteca fazendo o seu papel.


Criando um projeto em React.

O pré requisito para utilizar o React é ter o Nodejs com a versão estável mais atual instalado.

$ sudo apt-get update
$ sudo apt-get install nodejs
$ sudo apt-get install npm

Iniciando seu projeto:

Se já tem o React instalado, pule a etapa abaixo:
$ npm install -g create-react-app

Criando um projeto "my-app":
$ create-react-app my-app

Entre no projeto "my-app" e apague o conteudo da pasta "src":
$ cd my-app
$ rm -f src/*
