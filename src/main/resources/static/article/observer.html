Design pattern Observer.

O Observer é um padrão de projeto comportamental.
Esse tipo de padrão de projeto é utilizado quando se torna necessário a relação 1 para muitos objetos e os muitos objetos precisam ser notificados quando existe alguma alteração no objeto principal.
Um exemplo prático seria 3 interfaces gráficas diferentes que exibem as informações de formas diferentes mas utilizam 1 fonte de informação principal.


Diagrama exemplificando utilização do Observer.


Como funciona o Observer.

O pattern Observer descreve como estabelecer o relacionamento entre os objetos subject(assunto) e observer(observador).
Um Subject pode notificar vários Observer's que este ficará aguardando as atualizações.
O padrão Observer também é conhecido por Publisher-Subscriber.
