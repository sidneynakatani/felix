Reconhecimento de voz com Python.

Já não é novidade ferramentas de reconhecimento de voz.

A Microsoft tem a Cortana, a Apple possui a Siri e a Amazon a Alexa.

O Google também disponibiliza uma API de reconhecimento de voz para várias plataformas.

Essa semana minha filha ficou encantada com a possibilidade de se comunicar com um Macbook utilizando a Siri.

Como não tenho um Macbook, iniciei um projeto de sistema de reconhecimento de voz para o meu computador Linux.

Pesquisei um pouco sobre as Api's e resolvi utilizar a API do Python, pela facilidade de implementação.

Aproveitando esse projeto, vou demonstrar um tutorial de reconhecimento de voz com Python.

Iniciando seu reconhecimento de voz com Python.

Como estou utilizando uma máquina Linux para esse tutorial, o Python já vem embarcado na distribuição.

Se estiver utilizando Windows, instale antes o Python para continuar o tutorial.

Primeiro, vamos instalar a biblioteca speech recognition, digitando no terminal:

pip install SpeechRecognition

Agora vamos instalar a dependência de áudio pyaudio, digitando no terminal:

pip install pyaudio

Caso tenha problemas para instalar a biblioteca pyaudio como o erro “fatal error: portaudio.h”, tente os passos abaixo antes:

  1. Baixe os módulos: sudo apt-get install libasound-dev
  2. Baixe o pacote de áudio: http://portaudio.com/download.html .
  3. Descompacte o arquivo: tar -zxvf [portaudio.tgz]
  4. No diretório, digite: ./configure && make
  5. Finalmente, instale: sudo make install
