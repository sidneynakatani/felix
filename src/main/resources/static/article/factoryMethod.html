Design pattern Factory Method.

O Factory Method é um padrão de projeto de criação.
A motivação de utilização do padrão de projeto Factory Method é definir uma interface de criação de objeto e deixar as subclasses decidirem qual implementação instanciar.

O Factory Method permite centralizar a criação de objetos em somente uma classe que fica responsável por isso.

Podemos exemplificar a utilização do padrão Factory Method em um sistema que cria vários produtos diferentes.


Como funciona o Factory Method.

O pattern Factory Method exibe um design onde podemos centralizar uma fabrica de objetos.

Centralizando um criador de objetos, o código fica mais fácil de dar manutenção.
