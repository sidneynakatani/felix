Design pattern Memento.

O Memento é um padrão de projeto comportamental.
O padrão de projeto Memento é utilizado quando existe a necessidade de retornar o objeto ao estado anterior(Rollback ou Undo).

Projetos onde é necessário retornar o estado anterior como editores de texto ou banco de dados podem se beneficiar com esse padrão de projeto.

Um exemplo, seria um projeto de um editor de texto, onde é necessário desfazer o texto que foi digitado errado.
Utilizando o padrão de projeto Memento podemos retornar o estado do texto digitado anteriormente.


Como funciona o Memento.

O pattern Memento exibe um design onde trabalhamos com estado dos objetos.

Utilizando um Caretaker, controlamos todo acesso, armazenamento e recuperação dos estados dos objetos.

Todo objeto instanciado é armazenado em uma lista que fica disponível para o Caretaker. Quando existe a necessidade de voltar ao estado anterior, o Caretaker recupera da lista o estado anterior, remove da lista e retorna para o cliente.

Conclusão.
