Acessando API Bovespa com R.

Nesse Post vamos acessar os ativos da bolsa de valores Bovespa via API utilizando a linguagem R.

O R é muito utilizado por cientistas de dados devido a facilidade de manipular dados e execução de cálculos estatístico.

Vamos ver como é fácil utilizar o R e a biblioteca quantmod para manipular dados dos ativos da Bovespa e gerar gráficos.

Também vamos analisar os dividendos dos ativos com 1 linha.

O quantmod utiliza API de terceiros para apresentação dos dados dos ativos, e por essa razão algumas funcionalidades apresentam problemas.

Gerando gráficos dos ativos e seus dividendos.

- Primeiro precisamos instalar a dependência quantmod: install.packages("quantmod")

- Agora vamos configurar a biblioteca na sessão: library(quantmod)

- Vamos chamar a função, passando a sigla do ativo e a API do terceiro que vamos consultar, nesse caso a Yahoo e o Ativo da Petrobras: getSymbols("PETR4.SA",src="yahoo")

- Foi criado uma variável “PETR4.SA”, digitando no terminal podemos pegar as cotações do ativo: PETR4.SA

- Podemos gerar um gráfico de candlestick dos últimos dois meses: chartSeries(PETR4.SA, subset='last 2 months')

- Com uma simples função, podemos buscar todos os dividendo pagos: getDividends("PETR4.SA")

Conclusão.

Observe como o R apresenta bons recursos e desempenho para manipular dados, além da facilidade de gerar gráficos com comandos e funções simples.
