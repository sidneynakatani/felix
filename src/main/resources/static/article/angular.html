Aplicação Web Angular em 3 etapas.

Está cada vez mais fácil criar aplicações Web. Imagine criar uma estrutura para um projeto Web e subir essa aplicação com 3 comandos?
Várias linguagens de programação já oferecem esse recurso de Scaffold, mas nesse Post, vou apresentar utilizando Angular. Simples e rápido, não levando mais que alguns minutos para criar e subir a aplicação.
Vou usar a combinação Ubuntu 16 Lts, Nodejs 8 e NPM 5.


Linux + Nodejs + Angular 4.

Antes de começar o nosso projeto, precisamos verificar se estamos utilizando as versões compatíveis de Nodejs e Npm para iniciar:

  • Nodejs ≥ 6.9.x
  • npm ≥ 3.x.x

Agora só precisamos iniciar nosso projeto com algumas linhas de comando:

Se ainda não tem o Angular CLI instalado:
$ npm install -g @angular/cli

Vamos agora criar a estrutura do projeto:
$ ng new my-app




Se tudo ocorreu certo, entre na estrutura do projeto:
$ cd my-app




Agora vamos iniciar a aplicação:
$ ng serve -o





Conclusão.

Muito rápido e prático, mesmo instalando as dependências do Angular CLI não levou mais que 5 minutos na minha máquina.
