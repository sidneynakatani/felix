JustPy.

O JustPy é mais um framework para microsserviços Python.

A diferença desse framework é que a abordagem é voltada para embarcar back-end e front-end juntos.

Não acho que seja a melhor abordagem misturar as responsabilidades, mas acredito que para alguns casos onde precisamos construir aplicações web mais rápido pode ser interessante.

O JustPy aproveita as técnicas da Programação Orientado a Objetos, tornando o código mais limpo.

Outro ponto legal é que podemos utilizar componentes web junto com bibliotecas Python como pandas, matplotlib, entre outros.

Rodando Justpy.

Um pré requisito é que a versão do Python deve ser a versão 3 para rodar o Justpy.

Para instalar o Justpy, vamos utilizar o pip3:

$ pip3 install justpy
