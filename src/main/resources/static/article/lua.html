Lua Programming.

Recentemente recebi um placa de desenvolvimento NodeMCU programável em Lua. Comecei a ler a documentação dessa linguagem para iniciar essa programação e ver as especificações e features.
Fiquei surpreso com a potência dessa linguagem que não é tão conhecida(Pelo menos eu não conhecia muito) como Java, .Net e Python.
Essa linguagem é bem utilizada em engine de Games, segue um link com uma lista de engines Lua para games.


Um pouco sobre Lua.

A linguagem Lua foi criada integralmente no Brasil, na faculdade PUC-Rio(Pontifícia Universidade Católica do Rio de Janeiro).

Lua é uma linguagem leve e pequena. Sua versão Lua 5.3.4 tem apenas 1.1M e as suas bibliotecas completas 421K.

Apesar da documentação dizer que Lua pode ser programado Orientado a objetos e Funcional, para mim ela está mais para uma linguagem Procedural.
O código Lua é executada via interpretação de bytecodes para uma máquina virtual e possui coletor de lixo automático.



Portátil
Lua é distribuído em um pequeno pacote e pode ser descompactado e rodado em plataformas Windows e Unix, alem de dispositivos móveis como Android, iOS, BREW, Symbian, Windows Phone. Lua tambem roda em microprocessadores ARM e NodeMCU.


Opens Source
Lua tem seu código aberto, distribuido sobre a licença MIT, Lua pode ser usado inclusive para fins comerciais sem burocracias.


Embutível
Lua tem como característica embutir seu código com outras linguagens como C, C++, C#, Java, Ruby, Pearl e outras linguagens mais.
