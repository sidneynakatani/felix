Colaboratory.

O google Colab ou Colaboratory é um serviço nas nuvens que oferece uma máquina virtual com Python para incentivar aprendizado de Machine Learning e Inteligência Artificial.

Logicamente por trás desse serviço “Grátis” existe algum interesse da Google sobre o uso, provavelmente um aproveitamento dos algoritmos que são utilizados na plataforma.

A plataforma é bem interessante pois utiliza o Jupyter Notebook onde podemos combinar textos e códigos Python executáveis.

Uma ótima ferramenta para treinar Python sem precisar montar ambiente, além de poder compartilhar o workspace com amigos.

Outra forma de utilizar é montar uma aula de algoritmos, matemática e programação e distribuir para alunos.
