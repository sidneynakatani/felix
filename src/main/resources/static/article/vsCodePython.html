Visual Studio Code e Python.

Já não é de hoje que o Visual Studio Code é uma IDE muito utilizada pelos desenvolvedores.

O VS Code pode ser uma ótima opção de IDE para desenvolver projetos Python, leve, rápida e com várias extensões.

Já foi a época que o desenvolvimento Python era feito somente no bloco de notas.

Com as extensões Python para Visual Studio Code, podemos programar com opções de auto-complete e debugger da IDE.

Instalando extensão Python.

Para instalar a extensão Python para o Visual Studio Code, pressione “CTRL + p” e digite “ext install ms-python.python” ou baixe a instalação mais atual no marketplace da Microsoft .

Depois instale o Python lint para criticar o código em tempo de codificação:

>>> sudo /usr/bin/python -m pip install -U "pylint<2.0.0"

Agora podemos utilizar ferramentas de auto complete de código em tempo de codificação.

Também podemos utilizar a ferramenta de debugger.

Conclusão.

As IDE’s são ótimas ferramentas para aumentar a produtividade do desenvolvedor.
A Microsoft tem disponibilizado ótimas ferramentas de desenvolvimento e o VS Code é uma delas, com alto potencial e muito estável, a ferramenta tem agradado a comunidade de programadores.
