Spring + Visual Studio Code.

Parece não ter limites para o Visual Studio Code. Apenas instalando o pack Java que contempla extensão Java e bibliotecas para utilização do Maven e Spring já podemos subir uma aplicação Web em Spring Boot.

Cada vez mais fácil de criar, configurar e subir uma aplicação Web, ferramentas como GIT integradas ao VS Code e scaffolds Spring como Spring Initializr só vem para agregar e ficar.

Nesse Post vou mostrar como é fácil subir uma aplicação em Spring boot utilizando VS Code e Spring Initializr.
Se as extensões Java, Maven e Spring já estiverem instaladas, podemos subir uma aplicação em 3 minutos.


Baixando VS Code e extensão Java.

Se já tem o Visual Studio Code com as extensões Java instaladas, pule esse tópico.

Segue o Post anterior para fazer a instalação e configuração do Visual Studio Code.


Gerando estrutura incicial com Spring Initializr

O Spring Initializr é uma espécie de scaffold online. Com ele é possível gerar a estrutura inicial de um projeto Spring Boot.

Crie seu projeto inicial no Spring Initializr, baixe e descompacte na sua workspace.



1 - Selecione o tipo de Automatizador(Maven);
2 - Selecione a tecnologia(Java);
3 - Selecione a versão do Spring Boot(1.5.9);
4 - Digite o grupo(Pacote principal);
5 - Digite o nome do artefato;
6 - Selecione as dependências(Opcional).


Executando o Projeto

Selecione o Projeto que foi descompactado, File -> Open Folder e selecione a pasta principal.
