O.O com Python.

Orientação a Objetos(O.O) é um paradigma de programação que proporciona um significado para os programas criados através de atributos e métodos.

As linguagens de programação mais conhecidas que utilizam O.O hoje são Java, .Net e C++.

Python é uma outra linguagem de programação que podemos utilizar O.O para beneficiar deste padrão de programação.

Para quem programa em linguagens como Java, .NET e C++ vai sentir uma certa diferença na sintaxe.

Mas a estrutura de Classes em Python é bem fluente e de fácil entendimento da sua sintaxe.

Criando Classes com Python.

Para criar uma classe em Python utilizamos a palavra chave “class” e o nome da Classe.

Perceba um detalhe que o construtor é um pouco diferente de outras linguagens de programação: def __init__(), passando a referência da classe e seus atributos.

Outro detalhe é que não precisamos criar atributos privados para fazer referência no seu construtor.

Para instanciar a classe é bem simples, criando uma variável e indicando a classe com seus atributos.

Para trabalhar com herança em Python a sintaxe é um pouco diferente, onde passamos a classe Pai dentro dos parênteses.

Conclusão.

Uma pequena introdução de como utilizar Programação Orientada a Objetos com Python, mostramos como são as sintaxes de criação de classes e utilização de herança.
