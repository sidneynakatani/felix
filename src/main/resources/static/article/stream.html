API stream do Java 8.

Com a inclusão da API stream no Java 8, as tarefas como pesquisar e incluir condicionais em listas ficaram muito mais fáceis.

Já podemos tirar muito proveito utilizando stream, diminuindo o tamanho do código e deixando mais rápido.

Nesse artigo veremos alguns exemplos e como a API se comporta.

Primeiro, vamos entender alguns componentes da API stream:

  • map: utilizado para transformar um conjunto de itens em outros, por exemplo String para Integer;
  • filter: utilizado para filtrar ou pesquisar os itens da lista;
  • reduce: utilizado para reduzir o resultado, somando, subtraindo, média e outras reduções.

Com esses 3 métodos conseguimos manipular os dados sem muito esforço.

No começo teremos o esforço de aprender como utilizar recursos lambda, mas vale a pena como veremos utilizando a API.

Repare que utilizando stream, reduzimos muito o trabalho utilizando menos métodos de iterações como for, while e forEach.

Toda parte complexa de filtrar e transformar dados são feitos pela API, o que ajuda muito no desenvolvimento.
