Python e Números.

Não é segredo que para trabalhar com programação não é necessário ser um gênio da matemática, mas é necessário entender o básico de Álgebra .

De qualquer forma matemática e computação estão diretamente ligados.

Assim é importante entender como funcionam os números em Python.

Para especialistas em ciências de dados e inteligência artificial os números são essenciais para executar o trabalho.

Mas é importante que qualquer programador saiba utilizar os tipos corretos de números para cada solução.

Tipos numéricos em Python.

Em Python podemos trabalhar com 3 tipos numéricos: inteiros, ponto flutuante e números complexos.

Os números inteiros são basicamente tipos numéricos sem casas decimais, uma observação é que em Python 3 o inteiro não tem limitação de tamanho e o Python 2 existem o long-integer e integer.

Já os números com ponto flutuante são tipos numéricos que possuem casas decimais que podemos ajustar a precisão.

O último tipo é o número complexo, acredito que seja o tipo mais interessante pois são poucas linguagens de programação que trabalham tão bem com números complexos.

Com os números complexos para quem conhece álgebra, podemos trabalhar número real e imaginário.

Conclusão.

Sempre é importante entender como funciona as tipagens da linguagem de programação que está trabalhando, entender a tipagem numérica pode determinar a performance da sua aplicação.
