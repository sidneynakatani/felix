Kotlin, já conhece?

Já conhece Kotlin? Se não conhece, vale a pena parar alguns minutos e ler um pouco sobre essa nova linguagem de programação.
Uma linguagem funcional, Null-safety e interoperável com Java.
O Kotlin foi adicionado recentemente a lista de linguagem de progração para desenvolvimento Android, antes fazia parte dessa lista Java e C++.

Kotlin básico

Kotlin é uma linguagem de programação funcional que roda a partir de uma JVM, tornando uma linguagem totalmente interoperável com Java. Nesse mesmo contexto, podemos fazer um mix de códigos Java e Kotlin no mesmo projeto.
Uma linguagem consisa ou seja, clara e objetiva. Além de ser Null-safety, Type-safety e expressiva, tornando os códigos mais curtos.
Desenvolvida pela JetBrains, essa linguagem é muito parecida com Java e C#, tornando a curva de aprendizado para desenvolvedores Java e C# baixa.
As IDE's para desenvolvimento nesse momento são Eclipe e IntelliJ.
Para desenvolvimento Server-Side já estão disponíveis alguns frameworks de terceiros como Spring(a partir da versão 5), Vert.x, Ktor, kotlinx.html e conexão com banco via JDBC e JPA.


Propriedades dos campos "var" e "val"


Existem duas formas para declarar campos de propriedade, "var" onde você pode alterar seu valor e "val" onde o valor não pode ser mais alterado.

Exemplo:
var mutavel: Int = 1; // O valor pode ser alterado
val imutavel: Int = 2; // O valor não pode ser alterado



Tipos


Numérico:

Em Kotlin, tudo é um objeto. Diferente do Java onde podemos ter tipos primitivos também, acabando com problemas de conversão de tipos(autoboxing e unboxing).

Tipo Tamanho
Double 64
Float 32
Long 64
Int 32
Short 16
Byte 16

Em Kotlin, todos os numéricos fazem conversões de modo muito fácil e sem precisar de Cast explícito:

toByte(): Byte
toShort(): Short
toInt(): Int
toLong(): Long
toFloat(): Float
toDouble(): Double
toChar(): Char

Exemplo:
val i: Int = 1;
println("Resultado: "+ i.toDouble());
Resultado: 1.0



Caracter:

O tipo caracter é representado pelo tipo Char.

Exemplo:
val c: Char = 'a';



Booleano:

O tipo booleano é representado por Boolean.
Assim como Java, o booleano recebe como valores: true/false

Exemplo:
val v: Boolean = true;
val f: Boolean = false;



Arrays:

Arrays em Kotlin é representado pela classe Array.
Para criar um objeto tipo Array precisamos chamar a função arrayOf().

Exemplo:
val arrayDeString = arrayOf("a", "b", "c");



String:

A String em Kotlin assim como em Java, é imutável.


Representação com aspas duplas:
val s = "Olá, Kotlin!\n";

Representação com aspas triplas, ideal para textos longos:
val text = """
|Tell me and I forget.
|Teach me and I remember.
|Involve me and I learn.
|(Benjamin Franklin)
""";

String template:
val i = 100;
val x = "valor de i = $i";
println(x); // valor de i = 100



Condicional


Expressão If/Else:

Nada de diferente na expressão If em Kotlin

if (a > b) {  
      max = a  
} else {     
      max = b  
}        



Expressão When:

A expressão when substitui a expressão switch case

when (valor) {  
      1 -> print("valor = 1");
      2 -> print("valor = 2");
else -> {
      print("x não é 1 ou 2");  
}        



Laço For:

O laço for itera com qualquer coisa que provém de iterator

for (item in umaLista) {  
      print(item);
}        



Laço Do/While:

Nada de diferente na expressão Do/While em Kotlin

while (x > 0) {  
      print(x);  
}

do {  
      y++;  
} while (y != 10 );  





Null-Safety


Para fechar esse artigo, vamos falar sobre essa feature Null-Safety.
Kotlin, protege nosso sistema contra referencias nulas que acontece muito em Java. Mas e se tiver a necessidade de usar "null"? Não tem problema, pode-se usar "?" na frente da variável e a mesma fica apta a receber valores nulos.

Exemplo:
val nonNull: String = "teste";
val nullable: String? = null;
