Criar uma Wallet com Web3j.

O Web3j é uma biblioteca leve, rápida e reativa para Java e Android para trabalhar com contratos e integrar a redes Ethereum.

A ferramenta facilita a conexão com a rede Blockchain Ethereum.

Neste tutorial, vamos criar uma carteira para Ethereum e consultar o saldo.

Criando uma Wallet com Web3j.

Baixe a versão 3.4.0 do Web3j no link: https://github.com/web3j/web3j/releases/tag/v3.4.0 e descompacte em algum diretório de preferência.

Na pasta /bin digite: $ web3j wallet create

A aplicação vai solicitar que coloque uma senha e o diretório onde vai ser disponibilizado o arquivo, digite e guarde pois vai ser a senha da sua carteira de criptomoedas Ethereum.
